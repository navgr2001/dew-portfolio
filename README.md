# Interior Designer Portfolio

A futuristic and responsive portfolio website for an interior designer with 2+ years of experience.

## Tech stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- Motion for React

## Features

- Responsive layout for mobile, tablet, and desktop
- Sticky header with navigation menu
- Hero section with CV download button
- About, Projects, Skills, and Contact sections
- Smooth entrance, hover, and scroll animations
- Plain monochrome color system with futuristic styling
- Sample local assets included

## Setup

### 1. Install Node.js
Use Node.js 20 or newer.

### 2. Install dependencies
```bash
npm install
```

### 3. Start development server
```bash
npm run dev
```

### 4. Build for production
```bash
npm run build
```

### 5. Preview production build
```bash
npm run preview
```

## Project structure

```text
src/
  components/
  data/
  sections/
  App.tsx
  main.tsx
  styles.css
public/
  images/
  sample-cv.pdf
```

## Customization

- Replace content in `src/data/content.ts`
- Replace sample images in `public/images`
- Replace `public/sample-cv.pdf` with the real CV
- Connect the contact form to EmailJS, Formspree, or your own backend later
